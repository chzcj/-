## 1. 审核说明（复制到「版本描述 / 审核备注」）

```
育见是一款面向家长的 AI 家庭教育理解工具。

核心功能：
1. 日常交流：家长用文字或「按住说话」记录孩子当天情况，AI 流式回复理解与建议；
2. 沟通预演：模拟亲子对话场景，帮助家长练习沟通方式；
3. 任务跟进：将 AI 建议转化为可执行任务并记录反馈；
4. 孩子画像：基于家长输入生成结构化理解摘要（非医疗/诊断结论）。

测试账号说明：
- 使用微信一键登录即可体验完整流程；
- 首次使用需完成简短 onboarding（填写孩子基础信息）；
- 按住说话需授权麦克风，用于语音转文字输入。

隐私与合规：
- 已配置用户隐私保护指引；
- 麦克风仅用于按住说话时的本地录音与腾讯云 ASR 实时转写；
- 用户数据存储于 yujian.yihe.site 后端，不与第三方共享用于广告。

如有疑问请联系开发者。
```

## 2. 公众平台 · 用户隐私保护指引（建议文案）

在 [mp.weixin.qq.com](https://mp.weixin.qq.com) → 设置 → 服务内容声明 → 用户隐私保护指引 中填写：

### 收集的信息

| 信息类型 | 用途 | 是否必须 |
|----------|------|----------|
| 微信登录 openid | 账号识别与数据同步 | 是 |
| 麦克风录音 | 「按住说话」语音转文字（腾讯云 ASR） | 否，可改用文字输入 |
| 用户输入的育儿观察文本 | AI 理解与画像生成 | 是（核心功能） |

### 第三方 SDK / 服务

| 服务 | 用途 | 隐私政策 |
|------|------|----------|
| 腾讯云实时语音识别 | 按住说话转文字 | https://cloud.tencent.com/document/product/301/11470 |

### 开发者处理的信息

- 录音数据：实时发送至腾讯云 ASR 转写，转写完成后不长期保存原始音频（以实际后端策略为准）
- 对话与画像：存储于育见服务端，用于为用户生成个性化理解

### 用户权利

- 可在小程序「画像」页修改或注销账号（如已上线）
- 可拒绝麦克风权限，仍可使用文字输入

## 6. 常见驳回与应对

| 驳回原因 | 应对 |
|----------|------|
| 未配置隐私指引 | 按第 2 节填写，并确保 `__usePrivacyCheck__` 已开启 |
| 麦克风用途不明 | 审核备注说明「按住说话转文字，可拒绝后用文字」 |
| 类目不符 | 调整类目或弱化「诊断/治疗」表述 |
| 分享页空白 | 已全页配置 share hook，私密页落地登录 |
