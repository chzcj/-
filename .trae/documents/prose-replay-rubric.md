# Prose Replay 评分 Rubric（6 维 · 各 0–3 分）

> 用于 `scripts/replay-daily-prose.mjs` 产出报告；**禁止**用手工改写当 After。

## 维度

### 1. 档案咬合度（ArchiveFit）

| 分 | 标准 |
|----|------|
| 0 | 删掉 family 细节仍成立；或只复述最高频模板 |
| 1 | 有 family 词但与本题弱相关 |
| 2 | 1 条具体事实/原话与 **userText 本轮** 咬合 |
| 3 | 2+ 条且含 dossier/原话/一贯反应，明显「只有这家」 |

### 2. 答疑直接度（DirectAnswer）

| 分 | 标准 |
|----|------|
| 0 | 未回应家长本轮问题；或像产品经理验收话术 |
| 1 | 间接回应，绕圈 |
| 2 | 直接回应，但偏泛 |
| 3 | 像老教师当场答疑：先接住再问/再判，不念推理链 |

### 3. SecondMe 延续（Continuity）

| 分 | 标准 |
|----|------|
| 0 | 像首次见面 |
| 1 | 泛泛「之前聊过」 |
| 2 | 记得/认得其一 |
| 3 | 记得+认得+本次差量 |

### 4. 反模式（AntiPattern，反向计分：3=无问题）

| 分 | 标准 |
|----|------|
| 0 | 推理链口播；或整段复制感 |
| 1 | 客服/公文腔；或泛建议无 pack 支撑 |
| 2 | 轻微模板感 |
| 3 | 无上述问题 |

### 5. 单重点（SingleFocus）

| 分 | 标准 |
|----|------|
| 0 | 贪大全：多段结构、列点、换话题 |
| 1 | 2–3 个并列重点，略散 |
| 2 | 基本一个主答案，略有次要句 |
| 3 | 清晰只答本轮一个问题/一个重点 |

### 6. 说人话（PlainLanguage）

| 分 | 标准 |
|----|------|
| 0 | AI 套话（启动/卡的是/评价敏感/机制链） |
| 1 | 类型标签（「属于…类」）或报告腔 |
| 2 | 基本家常，略抽象 |
| 3 | 短句、动作/原话，念一遍就懂 |

## 总分

- **Sum** = 六维之和，满分 **18**
- 报告另列 **After 截断率**（`clampProse` 或末尾 `…`）

## 自动启发式（脚本内置，人工复核）

- 命中 `这里想区分` → AntiPattern ≤1
- userText 与 prose 无共有实词且 pack 非空 → ArchiveFit ≤1
- 句数 ≥6 或「首先/其次」→ SingleFocus ≤1
- 命中启动/卡的是/属于.*类 → PlainLanguage ≤1
