# 九州问学支教 · 育见公益试点推广材料包

面向支队长、学校老师的正式合作沟通材料。

## 文件清单

| 文件 | 用途 | 受众 |
|------|------|------|
| [01-项目技术说明.md](./01-项目技术说明.md) | 产品功能、技术架构、公益试点说明 | 支队长、学校老师、项目组内部 |
| [02-支队合作意向书.md](./02-支队合作意向书.md) | 合作内容、收益、流程、意向确认 | 支队长 |
| [03-支队长沟通话术.md](./03-支队长沟通话术.md) | 微信私聊、群内介绍、FAQ | 项目对接同学 |
| [assets/yujian-outreach-cover.png](./assets/yujian-outreach-cover.png) | 文档封面图 | 导出 PDF 时使用 |

## 使用建议

1. **私聊支队长**：先发 `03` 简短版话术 → 对方有意向再发 `01` + `02`
2. **对接学校**：打印或 PDF 发送 `01` 技术说明 + `04` 学校话术（在 `02` 第四节）
3. **家长群**：使用 `03` 第三节群内介绍稿

## 导出 PDF

```bash
# 若已安装 pandoc（可选）
pandoc docs/outreach/01-项目技术说明.md -o 育见-项目技术说明.pdf
pandoc docs/outreach/02-支队合作意向书.md -o 育见-支队合作意向书.pdf
```

或在飞书 / Notion / Typora 中打开 Markdown 后导出。
