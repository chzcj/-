## 一、合作背景

### 1.1 项目简介

**「育见」** 是清华大学 iCenter 联合教育领域团队推进的 AI 教育公益试点，核心产品为面向家庭教育场景的 **Web 端智能助手**。家长可通过文字或语音描述孩子在学习、成长、亲子沟通中的真实情况，系统基于多 Agent 深度建模与家庭记忆，提供比通用 AI 更贴近自家孩子的理解与支持建议。

> 产品体验地址：https://yujian.yihe.site  
> 技术说明详见：《育见 · 项目技术说明》（配套文档 01）

### 1.2 为什么找支队合作

- 支队与当地学校、家长有天然信任基础；
- 暑期支教是真实、高频的教育场景，适合公益试点收集反馈；
- 我们已在 **九州问学 30+ 支队** 及 **北京丰台区部分学校、社区** 逐步开展试点，希望与更多支队共建覆盖学生家庭的触达网络。

### 1.3 公益性质声明

请支队长与队员们放心：

| 承诺 | 说明 |
|------|------|
| ✅ 全程公益 | 不向家长、学校、支队收取任何费用 |
| ✅ 自愿参与 | 不强制家长注册或使用 |
| ✅ 非商业推广 | 不涉及产品销售、代理分成 |
| ✅ 尊重学校 | 建群、宣传形式均以学校意见为准 |
| ✅ 低工作量 | 不增加支队额外课业负担，可提供全套话术与材料 |

## 二、合作内容

### 2.1 我们希望支队协助的事项

**核心协助（可选，量力而行）：**

1. 若与当地学校沟通顺畅，协助联系学校老师；
2. 在学校同意的前提下，协助建立 **覆盖参与支教学生家长** 的交流群（微信等）；
3. 允许项目组在群内 **一次性或阶段性** 介绍公益试点项目，邀请有兴趣的家长自愿体验；
4. 日常可在群内发布支教动态（照片、活动记录等），增强家校互信。

**完全可选的进阶支持：**

- 向学校推荐项目组对接人，探讨长期合作（家庭教育讲座、AI 助学研讨等）；
- 协助发放简短问卷，收集家长对支教与产品的反馈。

### 2.2 项目组向支队提供的支持

| 支持项 | 内容 |
|--------|------|
| **全套材料** | 项目技术说明、宣传话术、FAQ |
| **群内介绍稿** | 可直接转发或改编的公益项目介绍 |
| **学校沟通话术** | 见配套文档 03 |
| **产品答疑** | 项目组在线解答家长与支队疑问 |
| **数据支持** | 在家长授权前提下，提供匿名化聚合反馈，供调研与答辩 |
| **品牌背书** | 可说明为「清华 iCenter 支持的教育 AI 公益试点」 |

## 三、对支队的实际收益

我们理解支队同学们可能担心「不熟、有顾虑、怕给学校添麻烦」。因此合作设计以 **支队自身也能受益** 为前提：

### 3.1 提升家校信任与支教满意度

- 家长群可用于发布 **孩子课堂照片、活动记录、每日小结**；
- 家长能更及时了解孩子在校情况，增强对支教工作的认可；
- 减少「孩子在学校怎么样我们不清楚」的信息落差。

### 3.2 便利实践调研与问卷

- 支队若需做 **家长满意度调查、支教效果问卷、访谈招募**，可直接在群内开展；
- 比临时逐个联系家长更高效、响应率更高。

### 3.3 支撑更出彩的实践成果

若家长自愿体验「育见」并授权反馈，支队可获得：

- **匿名化、聚合化** 的家庭成长关注点数据；
- 更具体的支教案例素材（如「家长普遍焦虑的作业拖延」「沟通断裂点」）；
- 支撑 **调研报告、支教总结、挑战杯/社会实践答辩** 的一手材料；
- 可选：将脱敏数据提供给当地学校，体现支队服务家校的额外价值。

### 3.4 积累高校公益项目联合实践经验

- 与清华 iCenter 支持的教育创新项目建立合作记录；
- 为个人简历、保研/求职材料增加 **真实公益项目协作** 经历；
- 表现突出的支队，项目组可出具 **合作证明 / 推荐信（需协商）**。

## 四、学校沟通建议话术

### 4.1 给学校老师的正式说明（推荐）

> 老师您好！
>
> 我们支队在暑期支教期间，除了开展日常课程教学外，也参与了一个 **清华大学 iCenter 联合教育团队开展的公益 AI 教育试点项目**（产品名「育见」），希望探索 AI 技术如何帮助家长更系统地理解孩子在学习、情绪、亲子沟通中的真实情况，提供家庭教育支持建议。
>
> 该项目目前已在部分学校和社区进行公益试点，**全程免费、自愿参与，不涉及收费，也不会影响学校正常教学管理**。
>
> 我们想请教：学校是否方便协助建立一个 **参与支教学生家长交流群**？后续我们可以在群里分享支教课堂动态、活动照片，同时邀请有兴趣的家长自愿了解这一公益项目。
>
> 具体形式完全尊重学校意见。若学校有负责家校沟通、德育或信息化的老师，也欢迎引荐给我们，项目组非常希望与当地学校建立长期交流，共同为学生家庭提供更好的成长支持。
>
> 谢谢老师！

### 4.2 简短版（微信沟通）

> 老师好～我们支队参与了一个清华 iCenter 支持的 **家庭教育 AI 公益试点**（育见），想问问学校方不方便协助建一个支教学生家长群，方便发课堂动态，也邀请有兴趣的家长自愿体验免费 AI 助手。全程公益、不强制，麻烦您看看是否可行～

### 4.3 学校可能关心的问题 · 标准答复

| 学校疑虑 | 建议答复 |
|----------|----------|
| 是否收费？ | 全程免费公益试点，无付费环节 |
| 是否影响教学？ | 不参与教学管理，仅家庭教育辅助 |
| 数据安全？ | 家庭数据隔离，对外仅匿名聚合；家长自愿授权 |
| 是否强制家长用？ | 不强制，群内介绍后自愿注册 |
| 能否只建群不推广 AI？ | 可以，建群本身对家校沟通就有价值 |

## 五、合作流程

```
支队表达意向 → 项目组发送材料包 → 支队与学校沟通
      → 学校同意建群 → 支队建群并邀请家长
      → 项目组发公益介绍（1 次或分阶段）→ 家长自愿体验
      → 支队可选：问卷/调研 → 项目组提供匿名数据支持
      → 支教结束：成果沉淀、可选出具合作证明
```

**预计支队额外时间成本：约 1–2 小时**（转发材料 + 与学校沟通 + 建群），其余由项目组承担。

---

## 六、合作原则

1. **学校优先**：任何动作以学校同意为前提；
2. **家长自愿**：不施压、不强制注册；
3. **支队无责**：因学校不同意而导致无法建群，完全理解，不影响其他合作；
4. **透明公益**：不向支队、学校、家长隐瞒项目试点性质；
5. **持续沟通**：项目组保持微信响应，随时答疑。
