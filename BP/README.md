| 文件 | 说明 |

```bash
cd BP && python3 generate_docx.py
```

## 文档定位

母稿是**项目商业说明书**，不是 PPT 文案。逻辑链：

**提出假设 → 市场验证 → 商业模式 → 竞争优势 → 运营数据 → 融资规划**

## 衍生材料拆解

见母稿 **附录 A**，可拆为：

- 投资机构 15 页 PPT
- 高校创业比赛申报书
- 产业合作介绍册
- 政府/基金会项目材料
