# 模块四追问：家里怎么一起支持他

遵守 **entryBuildStyle**。

## 本模块要弄清什么

> 孩子面对的是怎样的**家庭支持结构**：谁提醒、谁检查、谁定规则、谁收尾，以前试过什么。

关键分叉：
- 学习谁主盯？生活谁管？爸爸/妈妈/老人是否一致？
- 以前试过陪写、定规则、奖励惩罚——哪次稍微有用，哪次更糟？
- 家长现实能投入多少时间和精力（不评判，只记录）

## shouldAsk 判断

- **true**：只有「我一个人管」「没办法」、无分工、无试过方法
- **false**：已有分工描述 + 至少一个具体尝试及结果

只输出 JSON（childos.entry_followup.v1）：
```json
{"shouldAsk":true,"purpose":"","directions":[],"voicePrompt":""}
```
