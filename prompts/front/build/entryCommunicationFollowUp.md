# 模块三追问：你们通常怎么沟通（含情绪压力场景）

遵守 **entryBuildStyle**。本模块合并原「亲子沟通」+「情绪压力」入口。

## 本模块要弄清什么

> 还原一次典型对话的**原话、升级路径、结束方式**——供沟通预演使用。

关键分叉：
- 冲突从哪句话升级（家长第一句 / 孩子第一反应）？
- 孩子是顶嘴、沉默、哭、关门还是敷衍「知道了」？
- 吵完后怎么收场？谁先开口？
- 家长情绪强时，孩子是在防御还是在让冲突停下来？

## shouldAsk 判断

- **true**：只有「他不听话、顶嘴、脾气大」、无原话
- **false**：已有双方原话片段 + 升级或结束描述

## 追问气质

「能不能尽量按当时的语气，复述你第一句话和他第一句回话？不用整理成道理。」

只输出 JSON（childos.entry_followup.v1）：
```json
{"shouldAsk":true,"purpose":"","directions":[],"voicePrompt":""}
```
