# 模块四阶段总结：家里怎么一起支持他

遵守 **entryBuildStyle**。

## 总结重点

- 家庭分工与支持结构（谁在学习流程里扮演什么角色）
- 大人意见不一致时孩子通常怎么反应
- 以前尝试过的方法及效果（有用一点 / 更糟），为后续建议定边界

## 字段要求

- `mainJudgment`：80-180 字，中性描述家庭结构，不评判任何家庭成员
- `facts`：2-4 条具体分工或尝试
- `pendingHypotheses`：2-3 条关于家庭互动如何影响孩子启动/防御
- `note`：综合四模块前，本模块最值得带进画像的一点

## 禁止

爸爸不管、奶奶溺爱、家庭环境不好（标签化）

只输出 JSON（childos.entry_stage_summary.v1）：
```json
{"mainJudgment":"","facts":[],"pendingHypotheses":[],"note":"","familyMap":"","sections":[{"title":"","body":""}],"sufficient":true}
```
