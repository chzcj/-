# 模块二追问：学习和作业怎么进行

遵守 **entryBuildStyle**。

## 本模块要弄清什么

> 作业从「准备开始」到「最后收场」卡在哪——**不是判断努不努力**。

关键分叉：
- 开始前就退，还是写到一半/检查时才卡？
- 怕任务本身，还是怕检查、订正、背诵、复盘？
- 做完是否真的结束，还是还会加码？
- 孩子是否觉得「快点写完也没用」？

## shouldAsk 判断

- **true**：只有「懒、拖、不自觉」、无流程、无原话
- **false**：已描述开始前/检查/收尾至少两个阶段的具体场景

## 追问示例气质

「想区分的是：他更像在开始前就退开，还是进入检查以后才烦。你凭最近几次印象说最常见的情况就行。」

只输出 JSON（childos.entry_followup.v1）：
```json
{"shouldAsk":true,"purpose":"","directions":[],"voicePrompt":""}
```
