# 五、内容容器规范

## 5.1 卡片 Card

- **组件/场景**：信息卡片、商品卡片、内容展示卡片
- **适用场景**：内容聚合、商品展示、信息展示模块
- **官方标准规范**：卡片为独立内容容器，分割页面模块，自带统一圆角、留白。WeUI 中对应 `weui-panel`。
- **尺寸/参数**：
  - 卡片圆角：**12rpx**
  - 卡片内边距：**32rpx**
  - 卡片间距（模块外边距）：**24rpx**
  - 卡片左右外边距：**24rpx**（通栏卡片除外）
  - 轻微阴影：`0 2rpx 12rpx rgba(0,0,0,0.08)`
  - 卡片背景：`#FFFFFF`
- **交互规则**：整卡可点击或局部点击，点击反馈柔和（背景色微变或轻微缩放）。
- **强制要求**：卡片边界清晰，与页面背景（`#F6F6F6`）区分明显。
- **禁止设计**：
  - 卡片无留白、紧贴边界
  - 多层卡片嵌套杂乱
  - 卡片阴影过重
- **最佳实践**：所有同质信息统一卡片样式，保证页面整洁统一。商品卡片图文比例建议 1:1 或 16:9。

## 5.2 列表

- **组件/场景**：数据列表、设置列表、信息列表
- **适用场景**：功能入口、数据展示、设置页面
- **官方标准规范**：列表统一行高、间距，分割线统一。WeUI 中对应 `weui-cells` + `weui-cell`。
- **尺寸/参数**：
  - 列表单行高度：**80–90rpx**
  - 列表行左右内边距：**32rpx**（或 20rpx 适配不同场景）
  - 分割线位置：左侧对齐文字（无图标时贴左）或左侧缩进 32rpx（有图标时）
  - 分割线颜色：`rgba(0,0,0,0.1)`
  - 分割线厚度：**1px**（约 2rpx）
  - 列表图标：**40rpx × 40rpx**
  - 右侧箭头图标：**32rpx**，颜色 `rgba(0,0,0,0.2)`
  - 文字：**28rpx** 标题 + **24rpx** 辅助文字
  - Badge/提示文字：**24rpx**，颜色 `#FA5151` 或 `#888888`
- **交互规则**：列表点击整行反馈（`hover-class` 背景变灰 `rgba(0,0,0,0.05)`）；右侧箭头表示可跳转。
- **强制要求**：列表分割线统一，无错乱排版。单行列表高度 ≥ 88rpx。
- **禁止设计**：
  - 列表高度参差不齐
  - 无分割线或分割线颜色过深
  - 列表内容超出无截断处理
- **最佳实践**：功能入口优先使用宫格，数据展示优先使用列表。带图标的列表统一图标尺寸和左边距。

## 5.3 宫格 Grid

- **组件/场景**：功能入口宫格布局
- **适用场景**：首页功能入口、分类导航
- **官方标准规范**：WeUI 中对应 `weui-grids`。统一尺寸的网格布局，适用于功能图标 + 文字的入口展示。
- **尺寸/参数**：
  - 宫格常用列数：**2 / 3 / 4 列**（推荐 3 或 4 列）
  - 宫格单项高度：约 **160–180rpx**
  - 图标：**56rpx × 56rpx**
  - 文字：**24rpx**，居中对齐，图标下方
  - 图标与文字间距：**12rpx**
  - 宫格背景：`#FFFFFF`
  - 分割线：`rgba(0,0,0,0.1)`
- **交互规则**：点击整格跳转，有 hover 反馈。
- **强制要求**：列数固定（2/3/4），同一页面统一使用一种列数。
- **禁止设计**：
  - 宫格数量杂乱（一行 3 个、下一行 4 个）
  - 图标过大或过小
  - 无文字仅图标
- **最佳实践**：3 列或 4 列最常用，2 列适合功能较少的页面。每行数量固定，最后一行不足时留空。

## 5.4 折叠面板（Accordion）

- **组件/场景**：可展开/收起的折叠内容面板
- **适用场景**：FAQ、长内容分段展示、设置项分组
- **尺寸/参数**：
  - 折叠头部高度：**80rpx**
  - 标题文字：**28rpx**
  - 展开/收起箭头图标：**32rpx**，颜色 `rgba(0,0,0,0.4)`
  - 内容区内边距：**24rpx 32rpx**
  - 内容文字：**28rpx**
- **交互规则**：点击头部展开/收起，箭头旋转 180° 动效（0.3s transition）；支持手风琴模式（一个展开时其余收起）或多选模式。
- **强制要求**：折叠状态必须有明确视觉标识（箭头方向）。
- **禁止设计**：
  - 折叠无状态标识
  - 展开/收起动画卡顿
- **最佳实践**：默认全部收起，仅用户主动点击时展开。FAQ 场景适合手风琴模式。
